# Session Preparation

Read current context and active records, then create one `session-prep` record
with `python scripts/drydock.py new session-prep <id> --name "<name>"`.
Reference active situations, character concerns, faction actions, threats,
clues available through multiple paths, scarcity, and possible consequences.

Preparation is a draft, never canon and never a prescribed sequence. Ask the
Warden to review secrets, assumptions, and unresolved questions. Run validation
before play; rebuild context only if reviewed campaign state changed.
