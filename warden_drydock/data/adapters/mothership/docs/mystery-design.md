# Mystery Design

Write the underlying truth separately from evidence and interpretations.
Create `revelation`, multiple `clue`, and relevant `false-belief` records with
`python scripts/drydock.py new`. Give important information more than one
discoverable path and record source reliability and downstream implications.

All mystery records begin provisional and Warden-only. The Warden reviews the
truth, information paths, and what players currently know before canon status
changes. Validate the campaign and rebuild context after approved changes.
