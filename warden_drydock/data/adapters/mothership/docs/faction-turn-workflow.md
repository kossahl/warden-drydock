# Faction Turn Workflow

Create a `faction-turn` draft with `python scripts/drydock.py new`, then derive
actions from established faction motives, relationships, resources, and current
pressures. Record fictional actions, visible signs, affected clocks, and
proposed consequences without simulating proprietary procedures.

The Warden reviews every proposed world-state change before it becomes canon.
Apply approved updates to the affected records, validate, and rebuild context.
