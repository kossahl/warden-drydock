# Situation Design

Start from actors with incompatible motives, resources, scarcity, threats, and
a world that changes without intervention. Create or update `adventure`,
`faction`, `npc`, `location`, `ship`, `creature`, `clock`, and `consequence`
records using `python scripts/drydock.py new` for every new record.

Keep new material provisional. Review the situation, Warden truth, multiple
approaches, and likely escalation with the Warden. Do not prescribe scenes or
an ending. Validate after edits and rebuild context after approved canon or
current-state changes.
