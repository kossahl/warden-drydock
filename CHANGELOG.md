# Changelog

## 0.2.0 - Mothership 1e Narrative Adapter

- Expanded the Mothership adapter to 20 narrative record types covering the
  complete campaign and session lifecycle.
- Added declarative visibility, exact-value, contradiction, non-empty-field,
  and forbidden-heading validation, including strict player handout safety.
- Corrected canonical faction, NPC, and adventure paths with non-destructive
  legacy warnings and optional manual migration guidance.
- Added deterministic workflow guidance for campaign inception, situations,
  mysteries, sessions, faction turns, world state, canon, and handouts.
- Made adapter metadata authoritative across manifests, locks, upgrades, and
  validation while preserving standalone generated campaigns.

## 0.1.0 - MVP

- Added AI-assisted campaign bootstrap with standalone generated repositories.
- Added the Mothership adapter and declarative entity workflows.
- Added semantic validation and canon-safe, stable AI context generation.
- Added ownership locks and preview-first non-destructive upgrades.
- Added end-user, developer, adapter, and release documentation.
- Added wheel/source distribution verification and continuous integration.
